**About**

View at: `https://brickstreetdeli.pages.dev`

This project contains everything needed to run the website/webapp locally.

**Edit the code in your local development environment**

Any change pushed to the repo will also be reflected on the site, at the url mentioned above.

**Prerequisites:** 

1. Clone the repository using the project's Git URL
2. Navigate to the project directory
3. Install dependencies: `npm install`
4. Create an `.env.local` file and set the right environment variables

Run the app: `npm run dev`
